## 빌드 방법

### 1. 디렉토리 이동
```bash
cd /c/Users/이기준/Documents/OneDrive/바탕\ 화면/openbw-master/game_integrated/simple_rts
```

### 2. 컴파일
```bash
g++ -std=c++11 simple_game_advanced.cpp -lmingw32 -lSDL2main -lSDL2 -o simple_game_advanced.exe
```

또는 스크립트 사용:
```bash
chmod +x build_advanced.sh
./build_advanced.sh
```

### 3. 실행
```bash
./simple_game_advanced.exe
```

## 기능

- **유닛 선택**: 마우스 드래그로 선택
- **이동**: 우클릭으로 이동 명령
- **공격**: 적 유닛 우클릭으로 공격
- **A* 경로 찾기**: 장애물 회피 자동 경로 계산
- **고정소수점 연산**: OpenBW 방식의 fp8 사용
- **가속/감속**: 부드러운 유닛 이동

## 조작법

- **좌클릭 드래그**: 유닛 선택
- **우클릭**: 이동 또는 공격
- **ESC**: 종료
