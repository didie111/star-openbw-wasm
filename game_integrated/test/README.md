# StarCraft Clone - HTML Version

## 실행 방법

1. `index.html` 파일을 브라우저에서 열기
2. 또는 로컬 서버 실행:
   ```bash
   python -m http.server 8000
   ```
   그리고 `http://localhost:8000` 접속

## 조작법

### 마우스
- **좌클릭**: 유닛 선택
- **드래그**: 다중 선택 (박스 드래그)
- **Shift + 좌클릭**: 유닛 추가 선택
- **우클릭**: 이동 또는 공격 (스마트 명령)

### 키보드
- **A + 좌클릭**: 강제 공격 모드
- **S**: 선택된 유닛 정지
- **H**: 선택된 유닛 홀드 (제자리에서 공격만)
- **ESC**: 명령 모드 취소

## 구현된 기능

### 1. 유닛 시스템 (unit.js)
- ✅ 유닛 클래스 (HP, 쉴드, 공격력, 속도 등)
- ✅ 유닛 업데이트 (이동, 공격, 명령 처리)
- ✅ 3가지 유닛 타입 (마린, 질럿, 저글링)

### 2. 경로 찾기 (pathfinding.js)
- ✅ A* 알고리즘 구현
- ✅ 장애물 회피
- ✅ 최적 경로 계산

### 3. 충돌 감지 (collision.js)
- ✅ 유닛 간 충돌 감지
- ✅ 충돌 해결 (밀어내기)
- ✅ 위치 유효성 검사

### 4. 전투 시스템 (combat.js)
- ✅ 공격 범위 확인
- ✅ 데미지 계산 (공격력 - 방어력)
- ✅ 쉴드 시스템
- ✅ 공격 쿨다운
- ✅ 자동 공격 (범위 내 적 탐지)

### 5. 선택 시스템 (selection.js)
- ✅ 단일 유닛 선택
- ✅ 다중 유닛 선택 (드래그)
- ✅ Shift 추가 선택
- ✅ 선택 표시

### 6. 명령 시스템 (commands.js)
- ✅ 이동 명령
- ✅ 공격 명령
- ✅ 정지 명령
- ✅ 홀드 명령
- ✅ 스마트 명령 (우클릭)

### 7. 렌더링 (renderer.js)
- ✅ Canvas 2D 렌더링
- ✅ 유닛 그리기
- ✅ HP/쉴드 바
- ✅ 선택 표시
- ✅ 경로 표시

## 게임 로직 출처

이 코드는 OpenBW C++ 프로젝트의 핵심 로직을 JavaScript로 변환한 것입니다:

- **경로 찾기**: bwgame.h (9188-10745줄)
- **충돌 감지**: bwgame.h (1567-4437줄)
- **전투 시스템**: bwgame.h (869-890줄)
- **유닛 구조**: game_types.h (549-740줄)
- **유닛 데이터**: data_types.h (79-160줄)
- **명령 시스템**: actions.h

## 테스트 시나리오

1. **유닛 선택 테스트**
   - 마린 1개 클릭 선택
   - 드래그로 여러 마린 선택

2. **이동 테스트**
   - 선택된 유닛 우클릭으로 이동
   - 경로가 표시되는지 확인

3. **전투 테스트**
   - 마린 선택 후 질럿 우클릭 (자동 공격)
   - A키 누르고 질럿 클릭 (강제 공격)

4. **명령 테스트**
   - S키로 정지
   - H키로 홀드
